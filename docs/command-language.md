## Command Philosophy

Zu Beginn werden nur wenige Grundbefehle sichtbar angezeigt. Weitere Interaktionsmuster werden durch Exploration entdeckt und im Terminal/Field-Notes-Bereich freigeschaltet.

### Sichtbare Hauptbefehle
- umsehen
- gehe [richtung/ort]
- untersuche [objekt/ort]
- nimm [objekt]
- benutze [objekt] [ziel]
- hilfe / terminal

### Hidden Commands
Hidden Commands sind zusätzliche Verben, die Spieler durch Ausprobieren, Hinweise oder Storymomente entdecken. Sie erweitern das Gefühl, dass die Welt interaktiv ist.

Beispiele:
- öffne
- lies
- ziehe
- drücke
- drehe
- schiebe
- lausche
- kombiniere
- verbinde
- aktiviere
- deaktiviere

### Design-Regeln
- Hidden Commands sollen logisch aus der Situation ableitbar sein.
- Hidden Commands sollen möglichst wiederverwendbar sein.
- Einzelne Spezialaktionen werden bevorzugt über bestehende Commands gelöst.
- Neue Commands werden nur eingeführt, wenn sie das Explorationsgefühl stärken.
